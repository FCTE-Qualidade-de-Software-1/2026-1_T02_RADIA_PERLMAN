**0 segredos confirmados** no código-fonte (após triagem dos achados de alta entropia).

## Triagem

A varredura bruta retornou **2.497 achados de "High Entropy"** distribuídos em 206
arquivos distintos. A inspeção mostrou que **todos** são falsos positivos típicos de
alta entropia (hashes, bundles minificados, identificadores de build), **sem nenhum
segredo real**. Em particular, **nenhum achado** ocorre em `API/AcheiUnB/settings.py`
ou em qualquer arquivo de configuração sensível.

## Conclusão

Confirma-se que **não há segredos versionados** no repositório no snapshot avaliado. O
valor da métrica M1.1.1 é **0** (nível Aceitável, Fase 2 §5). Este resultado, combinado
com M1.1.2 (`.env` ignorado pelo Git), responde **Q1.1** de forma positiva.

## Sobre o dado bruto integral

O relatório JSON completo do trufflehog (~217 MB) **não é versionado no repositório** por
ser volumoso e composto exclusivamente de falsos positivos, o que o torna inadequado para
auditoria direta. Ele é **reproduzível** a partir do snapshot fixado com o comando abaixo;
o presente arquivo consolida e tria seu conteúdo para fins de rastreabilidade (F4-C2).

```bash
# Reexecução sobre o snapshot fixado (dados/00-snapshot.md)
trufflehog git file://. --branch e91773380c5259007d748d85e998a27362537339 --json \
  > M1.1.1_trufflehog_integral.json
```
