## Como auditar contribuições

- Cada seção tem, em seu rodapé, uma tabela de **Histórico de versão** com autor(es) e revisor(es).
- A página [Equipe](equipe.md) traz a tabela consolidada de contribuição da EU1.
- Cada *commit* substantivo na *branch* `main` inclui um bloco `Co-authored-by` no rodapé
  da mensagem, permitindo reconstruir a participação de cada integrante via
  `git log --format=%B` ou pela interface do GitHub.
